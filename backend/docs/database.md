# Arhitectura bazei de date 

# Schema bazei de date
Tabele principale:
- users (utilizatori)
- categories (categorii)
- food_items (produse alimentare)
- claims (revendicări)
- friends (prieteni)

# Relații intre tabele
- Un utilizator deține mai multe produse
- O categorie are mai multe produse
- Un produs poate fi revendicat de mai mulți utilizatori
- Un utilizator poate avea mai mulți prieteni
